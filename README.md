下载https://chromewebstore.google.com/detail/dhdgffkkebhmkfjojejmpbldmpobfkfo然后导入js文件

刷新https://i.cygnus.finance/页面右上角会出现一个黑色控制面板。

输入框：输入点击间隔时间（单位：毫秒，最少100）。

按钮：点击“开始”启动自动点击，再次点击“停止”暂停。
